📌 Project Overview

This repository contains the source code for my personal cybersecurity portfolio website.
Created as a school project, the website serves as a central hub to showcase my cybersecurity work—but more importantly, it represents who I am as a learner, builder, and future security professional.

The goal wasn’t just to list projects. I wanted to create a digital identity that feels professional, authentic, and aligned with my love for the cyber world.

🎯 Project Objectives (School Assignment)
Design and develop a fully functional personal portfolio website.
Demonstrate understanding of HTML, CSS, layout, responsiveness, and UI choices.
Showcase at least six security-related projects with descriptions, tools, and links.
Express personality, originality, and creativity in the site’s design.
Maintain readability, accessibility, and smooth navigation.

Challenge	Solution

Showing personality while staying professional	Used subtle visual identity + clean writing
Organizing projects in a readable way	Used consistent card structure + tool icons
